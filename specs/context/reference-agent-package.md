# Context: Reference Agent Package

> Generated by `/spec-bridge` Stage 0. Evidence file for `specs/feature-intake/reference-agent-package.md`.

## Task Intent

- Extract the current quickstart built-in LLM/stub brain into a new `@facet/reference-agent` package.
- Keep `@facet/quickstart` as the one-command CLI/server/page-bundle wrapper that composes the reference agent.
- Preserve current quickstart behavior and provider/stub safety; do not implement agent quality improvements in this change.
