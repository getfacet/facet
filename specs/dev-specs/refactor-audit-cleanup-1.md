## Fail-safe & Boundary Checklist

- Malformed / empty / null input: CLI `theme` with missing/empty/invalid name
  throws a descriptive error before patch emission; persistent `theme(name)` tool
  returns an error tool result and records no stage op.
- Deeply nested / cyclic tree: unchanged; this work does not alter tree
  validation or rendering.
- Offline agent / rapid events / racing agent turn: unchanged; theme commands are
  ordinary ordered stage patches, and bridge event serialization remains as-is.
- Invalid theme name: rejected at local action surfaces using `isValidThemeName`,
  preventing live/stored divergence.
- Raw CSS values: rejected by the same name validator because values with spaces,
  punctuation, `#`, parentheses, or CSS syntax are not valid theme names.
